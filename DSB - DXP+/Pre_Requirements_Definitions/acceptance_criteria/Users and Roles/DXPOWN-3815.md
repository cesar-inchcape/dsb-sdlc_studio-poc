3PD Digital Service Booking - 3PD Autum + Americar | Retail Group User With Multi‑Workshop Access and Configurable Visibility


AS a retail group user
I WANT access to multiple workshops with permissions tailored to my profile
SO THAT I can operate only within the areas assigned to me.

Acceptance Criteria
AC1 – Multi‑Workshop Assignment
GIVEN a retail group user is configured
WHEN accessing their scope
THEN the user MUST be allowed to access multiple workshops belonging to the retail group.

AC2 – Custom Function Visibility
GIVEN the user accesses a module
WHEN their permission is checked
THEN the system MUST enable or hide features based on assigned permissions:

Create

Edit

View

Delete

AC3 – Admin Panel for Permissions
GIVEN a system admin configures user profiles
WHEN adjusting permissions for retail group users
THEN they MUST be able to toggle visibility of:

Reporting

Workshop management

Service management

Appointment operations
